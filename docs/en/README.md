# # Vue-Antd-Admin

## Project setup
### Clone the project
```
git clone https://github.com/KXinghui/vue-antd-admin.git
```  

### Install dependency
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Lints and fixes files
```
npm run lint
```
