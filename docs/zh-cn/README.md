# Vue-Antd-Admin

## 项目设置
### 克隆项目
```
git clone https://github.com/KXinghui/vue-antd-admin.git
```  

### 安装依赖
```
npm install
```

### 编译和热重装以进行开发
```
npm run serve
```

### 编译并最小化生产
```
npm run build
```

### 运行单元测试
```
npm run test:unit
```

### 整理和修复文件
```
npm run lint
```

### 自定义配置
请参阅 [配置参考](https://cli.vuejs.org/config/).
